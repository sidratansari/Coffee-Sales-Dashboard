☕ Coffee Sales Dashboard

1. Project Title / Headline

☕ Coffee Sales Analytics Dashboard – Sales, Profit & Customer Insights
An interactive Power BI dashboard designed to analyze coffee shop sales performance, customer purchasing behavior, product demand, payment preferences, weather impact, and profit trends across multiple store locations.

2. Short Description / Purpose

The Coffee Sales Dashboard provides a comprehensive view of sales, profits, customer behavior, and product performance across coffee stores. It helps business owners and managers identify top-performing products, optimize operations, and make data-driven decisions to increase revenue and customer satisfaction.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Used for creating interactive reports and visualizations.

🔄 Power Query – Data transformation and cleaning.

🧠 DAX (Data Analysis Expressions) – Created KPIs, calculated measures, and business logic.

🗂️ Data Modeling – Established relationships between sales, customer, and product data.

📁 File Format – .pbix for dashboard development and .png for dashboard previews.

4. Data Source
Source: Coffee Shop Sales Dataset

The dataset contains information related to:
•Orders and transactions
•Product categories
•Store locations
•Customer types
•Payment methods
•Weather conditions
•Sales and profit metrics
•Monthly business performance
•Product quantity sold

5. Features / Highlights

📌 Business Problem

Coffee chains generate large volumes of sales data daily, making it difficult to understand customer preferences, product demand, and profitability across locations.

Key questions include:
•Which coffee products sell the most?
•Which stores generate the highest profits?
•How do weather conditions affect sales volume?
•What payment methods do customers prefer?
•How does profit fluctuate throughout the year?

🎯 Goal of the Dashboard
To provide an interactive analytics solution that:

Monitors sales and profit performance.
Identifies best-selling products.
Analyzes customer purchasing behavior.
Evaluates store profitability.
Understands the impact of weather and seasonal trends on sales.
